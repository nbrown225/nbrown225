## Christina Brown's Portfolio 👋

### This project is to create a clean, professional looking portfolio th at will set to public on my github

### What is needed?
* Name
* Recent photo
* Links/sections to recent work/projects
* How to contact me
* Navigation and links that lead to sections
* Images of recent work - first image should be bigger than others
* Clicking on image of applications should bring you to the deployed application
* Layout must be responsive
* 
### Applications and websites used
* VS Code
* GitBash
* Github
* Google Slides (for planning out website structure)
*

### New Tricks Learned
* calc() -function in css that calculates the value. can be used anywhere
* linear-gradiant -fades the background to different colors
* list-style:image -allows you to add an image to a list instead of a bullet point
* <a href = "mailto: abc@example.com">Send Email</a> - when clicking on the email link, it opens the email app to send to that specified email
